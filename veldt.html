Veldt Encounter Guide

Finding Specific Enemies

To find a specific enemy on the Veldt, follow these steps:

  1. Figure out which formation group below contains the enemy that you want.
  2. Fight on the Veldt and determine which group contains the encounter you're on.
  3. Repeat (2) until the NEXT group contains the enemy you want.
  4. Perform an emulator save state (not in-game save state).
  5. Fight a battle on the Veldt. If it contains your enemy, great! You're done!
  6. If it doesn't, load the save state.
  7. Fight an encounter off-Veldt.
  8. Return to the Veldt.
  9. Repeat steps 4-8 until you find your target enemy.

For example, I'm trying to find HellRidr, which I can see from using "CTRL-F/Find" that it's only in formation #10. I go onto the Veldt and encounter a solo SoulEatr, so I know that I'm in formation #61. I fight 12 more battles after that, so my last battle was an encounter in formation #9 (62, 63, 0-9). I perform a save state, then fight a battle on the Veldt, but run into 3x LivingDe. That's not what I want, so I reload the save state, then go fight a battle off of the Veldt. I come back and, luckily, encounter HellRidr on my first battle afterwards. I can now cross HellRidr off of my Rages list.

Veldt Mechanics

  1. Every time you run into an encounter on the Veldt, the following occurs:
    • (a) Increase "group number" by 1.
    • (b) If the group number is 64 or higher, subtract 64 until it's below 64 (or ID%64).
    • (c) Select a random encounter from the list in the new group.
    • (d) If you have not encountered that formation, pick the next one in the group.
    • (d) If it's not a valid Veldt encounter, pick a different one in the same group.
    • (e) If you have not encountered ANY enemies in a group, return to (a).
  2. Whenever you get into an encounter off the Veldt, the "randomly chosen" encounter from the current pack is changed.
  3. Every time you load an in-game save, the system/global save's RNG is increased by 13 and immediately saved, then the group ID is set to that. This may or may not match up with your save's previous group ID+13, and thus is only a semi-reliable way to increase the group ID by a fixed amount. I would not rely on this method to find a particularly stubborn enemy, but if you're lucky, the stars may align and it could save you time, or at least pick a random formation much closer to the one you're going for.
  4. The memory address [7E1FA5] contains last encountered formation group after a battle.
